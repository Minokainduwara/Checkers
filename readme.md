# GuiNN Checkers

GuiNN Checkers is a modern, cross-platform desktop checkers game developed in **C** using the **SDL2** multimedia library. The project aims to demonstrate professional software engineering practices while delivering a polished and enjoyable gaming experience.

Unlike a traditional academic checkers project, GuiNN Checkers is designed with a modular architecture, reusable UI components, smooth animations, customizable themes, and an AI opponent. The codebase is structured for maintainability, scalability, and portability across multiple operating systems.

## Features

- Modern graphical user interface (GUI)
- Cross-platform support (macOS, Linux, Windows)
- Human vs Human gameplay
- Human vs AI gameplay
- Minimax AI with Alpha-Beta Pruning
- Multiple board and piece themes
- Save and load game functionality
- Player statistics and achievements
- Background music and sound effects
- Smooth animations and transitions
- Responsive UI system
- Modular game engine architecture
- CMake-based build system
- Clean, documented, and maintainable C code

## Technology Stack

- Language: C17
- Graphics: SDL2
- Image Loading: SDL2_image
- Font Rendering: SDL2_ttf
- Audio: SDL2_mixer
- Build System: CMake
- Version Control: Git

## Project Goals

- Develop a complete desktop game using modern C programming practices.
- Build a reusable 2D game framework using SDL2.
- Implement the complete rules of international checkers.
- Create an intelligent AI opponent using Minimax and Alpha-Beta Pruning.
- Design a clean, responsive, and visually appealing user interface.
- Maintain a clean architecture suitable for future extensions.
